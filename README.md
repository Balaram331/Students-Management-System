# 🎓 Student Management System

A modern desktop-based Student Management System built using Python, Tkinter, and SQLite to simplify student record management for schools and educational institutions.

The system provides an easy-to-use interface for managing students, tracking attendance, recording marks, generating results, and analyzing academic performance.

---

## 🚀 Why This Project Was Built

Managing student records manually can be time-consuming, inefficient, and prone to errors.

This project was developed to digitize and automate the entire student management process by providing a centralized system for:

- Maintaining student information.
- Recording attendance efficiently.
- Managing subject-wise marks.
- Generating results automatically.
- Visualizing academic performance through analytics.

The goal is to provide an organized, reliable, and user-friendly academic management solution.

---

##  Features

###  Student Management
- Add new students.
- View all student records.
- Update student details.
- Delete student information.

###  Attendance Management
- Record daily attendance.
- Mark students as Present or Absent.
- Store attendance history in the database.

### Marks Management
- Add subject-wise marks.
- Store and manage academic records.

### Result Generation
- Automatically calculate:
  - Total Marks
  - Percentage
  - Grade
- Generate student report cards.

### Analytics Dashboard
- Total student statistics.
- Average marks analysis.
- Highest marks tracking.
- Pie chart visualization.
- Subject-wise performance graphs.

### PDF Report Generation
Generate professional PDF report cards for students.

### Authentication System
Secure login system for authorized access.
